## 🎰 Slot Machine Game (Python)

### About
This is a simple command-line Slot Machine game built using Python. The game allows users to deposit money. place bets, spin the slot, and win and the winning are added to balance.

### Features
- Deposit money to start playing
- Choose number of lines to bet on
- Set bet amount per line
- Randomized slot spins
- Win calculation based on matching symbols
- Displays current balance after each round
- Used Python 3.10.5

### To play ▶

1. Clone or download this repo.
2. Open terminal/command prompt
3. Navigate to the project folder
4. Run the program:

### How to Play

- Enter the amount you want to deposit

- Choose how many lines to bet on

- Enter your bet per line

- Spin the slot machine

- Win if symbols match across selected lines
